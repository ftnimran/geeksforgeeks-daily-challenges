# Check Preorder of BST

🔗 [Problem Link](https://www.geeksforgeeks.org/problems/preorder-traversal-and-bst4006/1)
📊 Difficulty: Medium
📂 Category: General

## 📝 Description

Given an array arr[ ] consisting of distinct integers, check if the given array can represent preorder traversal of a BST.

**Examples :**

```
Input: arr[] = [2, 4, 3]

Output: true
Explaination: Given arr[] can represent preorder traversal of following BST:
```

![Problem Image](https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/929468/Web/Other/blobid0_1780982743.webp)

```
Input: arr[] = [2, 4, 1]

Output: false
Explaination: Given arr[] cannot represent preorder traversal of a BST.
```

**Constraints:**  
1 ≤ arr.size() ≤ 10<sup>5</sup>  
0 ≤ arr[i] ≤ 10<sup>5</sup>
